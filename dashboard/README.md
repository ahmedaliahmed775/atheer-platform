# لوحة تحكم Atheer

> لوحة تحكم إدارية لمراقبة وإدارة منصة الدفع اللامسي

## 📋 الوظيفة

الداشبورد تخدم طرفين:
- **فريق Atheer** — إدارة شاملة: محافظ، مستخدمين، إعدادات، تقارير
- **ممثلو المحافظ** — عرض معاملات وإحصائيات محفظتهم فقط

## 🖥️ الشاشات (8 شاشات)

| الشاشة | المسار | الوصف |
|--------|--------|-------|
| لوحة القيادة | `/dashboard` | ملخص: معاملات اليوم، نسبة النجاح، حالة المحافظ |
| المعاملات | `/transactions` | بحث وفلترة + تصدير Excel |
| تفاصيل معاملة | `/transactions/{id}` | تدفق كامل مع timeline |
| المستخدمون | `/users` | إدارة المسجلين (تعليق/إلغاء/تعديل حدود) |
| المحافظ | `/wallets` | إضافة/تعديل/تعطيل محفظة |
| الإحصائيات | `/analytics` | رسوم بيانية (حجم، نجاح/فشل، ذروة) |
| التسوية | `/reconciliation` | مقارنة سجلات السويتش vs المحفظة |
| الإعدادات | `/settings` | حسابات الداشبورد، تدوير مفاتيح |

## 👥 الأدوار

| الدور | الصلاحيات |
|-------|-----------|
| `SUPER_ADMIN` | كل شيء |
| `ADMIN` | إدارة بدون تدوير مفاتيح |
| `WALLET_ADMIN` | قراءة بيانات محفظته فقط |
| `VIEWER` | قراءة فقط |

## 🚀 التشغيل

```bash
npm install
npm run dev    # http://localhost:3000
```

## 🛠️ التقنيات

- **Framework:** Next.js 14+ (App Router)
- **UI:** shadcn/ui + Tailwind CSS
- **Charts:** Recharts
- **Tables:** TanStack Table
- **Auth:** JWT (httpOnly cookies)
- **Export:** xlsx

## 🔗 التكامل

الداشبورد تتصل بالسويتش عبر Admin APIs فقط (`/admin/v1/*`).
لا تتصل بقاعدة البيانات مباشرة.
